## Авторизация пользователя

### Первый тест
Предусловие: Пользователь зарегестирован в системе

Шаги:

1. Ввести валидный логин
2. Ввести валидный пароль
3. Нажать на кнопку "Login"

Ожидаемый результат: Авторизация пройдена, пользователь перенаправлен на главную страницу: https://www.saucedemo.com/inventory.html 

### Второй тест

Шаги:

1. Ввести валидный логин
2. Ввести невалидный пароль 
3. Нажать на кнопку "Login"

Ожидаемый результат: Пользователь получит окно с ошибкой:
"Epic sadface: Username and password do not match any user in this service"
